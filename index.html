phpBB-Auto-Database-Backup

An extension for phpBB 3.1 that will automatically backup your board database using the phpBB 3.1 Cron System.

View the Project on GitHub Pico/phpBB-Auto-Database-Backup

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Features

  • Enable/disable Auto Database Backup
  • Backup frequency
  • Database optimization before backup
  • The quantity of stored backups
  • The backup time
  • The backup file type - gzip, bzip2 or text (depends on server settings)

Requirements

phpBB 3.1.0 RC3 or newer recommended.
